# homebrew-phs

Homebrew tap for [PHS CLI](https://github.com/NethermindEth/angkor-platform-api) — the Platform Hosting Services command-line tool.

## Install

```bash
brew tap NethermindEth/phs
brew install phs
```

## Upgrade

```bash
brew upgrade phs
```

## Uninstall

```bash
brew uninstall phs
brew untap NethermindEth/phs
```
